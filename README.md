# FexFlasher

A Windows tool to flash `.fex` firmware files to SD cards.

## Installation
Download the latest release from the [Releases page](../../releases).

## Usage
1. Insert your SD card.
2. Run the `.exe` file.
3. Select your `.fex` file and flash.
